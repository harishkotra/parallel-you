### Fixed

### Added
- Chrome extension (Manifest v3) with persona-based reply generation.
- Predict, Truth Mode, Shadow User (opt-in), and Reply Later UI/actions.
- Cloud Run backend with OpenAI-powered endpoints.
- Hosted web app, downloadable extension package, and privacy policy page.
