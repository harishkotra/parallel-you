Parallel You Privacy Policy

Effective date: April 25, 2026

Parallel You is a browser extension and companion web app that helps users draft social-media replies with AI. This policy explains what data we process, why we process it, and how users control it.

What We Collect

  • Text you submit for AI features (reply generation, prediction, truth mode, and reply later).
  • If you opt in to Shadow User mode, interaction text and typed text snippets from supported platforms.
  • Basic local preferences stored in browser extension storage, such as opt-in state and API URL.

How We Use Data

  • To generate AI responses you explicitly request.
  • To analyze draft tone, engagement, and risk before posting.
  • To power optional asynchronous reply suggestions.
  • To remember extension settings across sessions.

Data Sharing

We do not sell personal data. We send requested text inputs to our AI processing provider (OpenAI) to return model outputs. Data may also be processed by cloud infrastructure providers used to host the service.

User Controls

  • Shadow User tracking is opt-in and can be turned off at any time.
  • The extension does not auto-post content to social platforms.
  • You remain in control of whether to use or post generated output.

Storage and Retention

Current demo deployment uses lightweight in-memory storage for behavioral and scheduled-reply data. Data is not intended for long-term persistence in this demo setup and may be cleared on service restarts.

Security

API keys are managed as server environment variables and are not exposed in the extension package. We apply reasonable safeguards appropriate for a demo-stage service.
